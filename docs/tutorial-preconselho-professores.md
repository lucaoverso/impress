# Tutorial do Módulo de Pré-Conselho para Professores

Este guia foi preparado para orientar o professor em toda a jornada de uso do módulo de pré-conselho, desde o acesso ao sistema até o salvamento e a revisão dos registros.

## 1. Objetivo do módulo

O módulo de pré-conselho foi criado para:

- registrar estudantes que precisam de atenção pedagógica em uma turma e disciplina;
- padronizar os motivos do apontamento;
- gerar automaticamente um parecer inicial por estudante;
- organizar os registros para posterior consolidação pela coordenação.

## 2. Antes de começar

Para usar o módulo corretamente, é importante que:

- exista um período de pré-conselho cadastrado;
- o período esteja com status `Aberto`;
- o professor esteja vinculado às turmas e disciplinas corretas em sua carga docente.

Se uma turma ou disciplina não aparecer para você, normalmente isso indica que a atribuição docente ou o período precisa ser ajustado pela gestão.

## 3. Como acessar

1. Entre no sistema com seu usuário e senha.
2. Acesse a tela de `Serviços`.
3. Clique no cartão `Pré-Conselho`.
4. A aba principal para o professor será `Meu preenchimento`.

## 4. Visão geral da tela do professor

Na aba `Meu preenchimento`, você encontrará quatro áreas principais:

- `Período`: local para escolher o bimestre ou período de trabalho.
- `Turmas do professor`: lista das combinações de turma e disciplina disponíveis para sua carga.
- `Lista de estudantes`: relação dos estudantes da combinação selecionada.
- `Registro individual`: janela de preenchimento do parecer por estudante.

Também aparecem três indicadores no topo do painel:

- `Combinações na carga`: quantas combinações de turma e disciplina estão disponíveis.
- `Estudantes sinalizados`: quantos já possuem registro no período selecionado.
- `Pendências`: quantos ainda não foram sinalizados.

## 5. Jornada completa do professor

### Etapa 1. Selecionar o período

1. No campo `Período`, escolha o período desejado.
2. Clique em `Atualizar painel`.

O sistema recarrega os dados com base no período selecionado.

Se o período estiver fechado:

- os dados ainda poderão ser consultados;
- o professor não conseguirá editar nem salvar novos registros.

### Etapa 2. Escolher a turma e a disciplina

1. Na coluna `Turmas do professor`, localize a combinação desejada.
2. Clique na turma/disciplina que você quer preencher.

Ao selecionar uma combinação, o sistema carrega:

- os estudantes daquela turma;
- os registros já salvos para aquela disciplina no período.

### Etapa 3. Localizar o estudante

Na coluna `Lista de estudantes`, você pode:

- usar o campo `Buscar estudante` para digitar parte do nome;
- usar o filtro para visualizar:
- `Todos`
- `Sinalizados`
- `Não sinalizados`

Quando encontrar o estudante, clique para abrir a janela de preenchimento.

### Etapa 4. Abrir o registro individual

Ao clicar no estudante, abre-se a janela `Parecer por estudante`.

Nela você verá:

- o nome do estudante selecionado;
- a indicação se já existe um registro salvo;
- os motivos padronizados, organizados por categoria;
- o campo `Nível de atenção`;
- o campo `Observação complementar`;
- a área `Pré-visualização do parecer`.

## 6. Como preencher um registro

### 6.1 Selecionar os motivos

Os motivos são a base do parecer automático.

1. Marque um ou mais motivos que descrevem a situação do estudante.
2. Os motivos estão agrupados por categorias, como:
- avaliação;
- participação;
- comportamento;
- frequência;
- organização e estudo;
- dificuldades pedagógicas.

Importante:

- é obrigatório selecionar pelo menos um motivo para salvar;
- sem motivos selecionados, o parecer não é gerado.

### 6.2 Definir o nível de atenção

No campo `Nível de atenção`, selecione:

- `Baixo`
- `Médio`
- `Alto`

Esse campo ajuda a orientar a recomendação pedagógica automática incluída no texto final.

### 6.3 Registrar uma observação complementar

No campo `Observação complementar`, escreva informações adicionais que ajudem a contextualizar o caso.

Exemplos:

- dificuldades persistentes na entrega de atividades;
- necessidade de retomada de conteúdos essenciais;
- pouca autonomia na rotina de estudos;
- oscilação de participação ao longo do período.

Use esse campo para complementar, não para repetir os motivos já marcados.

### 6.4 Conferir a pré-visualização

À medida que você:

- marca os motivos;
- altera o nível de atenção;
- digita a observação complementar;

o sistema atualiza automaticamente a `Pré-visualização do parecer`.

Essa área mostra como ficará o texto gerado para o estudante antes do salvamento.

## 7. Como salvar

Depois de conferir o texto:

1. Clique em `Salvar registro`.
2. O sistema salvará o parecer e aplicará a sinalização do estudante automaticamente para aquela turma, disciplina e período.

Após o salvamento:

- o estudante passa a constar como sinalizado;
- o registro pode ser revisto depois;
- o painel é recarregado com os dados atualizados.

## 8. Como editar um registro já salvo

Se o estudante já tiver registro:

1. localize o estudante novamente na lista ou na relação de registros;
2. abra o registro;
3. ajuste motivos, nível de atenção ou observação;
4. clique em `Salvar registro` novamente.

O sistema atualizará o parecer já existente.

## 9. Como excluir um registro

Se um registro tiver sido lançado indevidamente:

1. abra o estudante com registro salvo;
2. clique em `Excluir registro`;
3. confirme a exclusão.

Também é possível excluir por ações disponíveis na lista de registros já salvos, quando exibida.

Atenção:

- a exclusão remove o registro daquele estudante para a combinação selecionada;
- essa ação deve ser feita com cuidado, especialmente perto da consolidação final.

## 10. Como limpar o formulário

O botão `Limpar formulário` serve para:

- desmarcar os motivos;
- limpar a observação;
- limpar a seleção atual do registro.

Use esse botão quando quiser recomeçar o preenchimento antes de salvar.

## 11. O que acontece quando o período está fechado

Quando o período estiver com status `Fechado`:

- o professor ainda pode consultar os dados;
- o formulário fica indisponível para edição;
- não é possível salvar nem excluir registros.

Se precisar alterar algo em período fechado, solicite à coordenação ou administração a reabertura do período.

## 12. Boas práticas de preenchimento

- Escolha motivos objetivos e coerentes com o que foi observado em sala.
- Evite marcar muitos motivos sem necessidade.
- Use a observação complementar para contextualizar, não para repetir.
- Revise a pré-visualização antes de salvar.
- Registre apenas casos que realmente precisam de acompanhamento no pré-conselho.
- Mantenha a escrita profissional, clara e respeitosa.

## 13. Exemplos de fluxo de uso

### Exemplo 1. Primeiro registro de um estudante

1. Selecionar o período.
2. Escolher a turma e disciplina.
3. Abrir o estudante.
4. Marcar os motivos.
5. Definir o nível de atenção.
6. Escrever a observação complementar.
7. Revisar a pré-visualização.
8. Salvar.

### Exemplo 2. Atualizar um estudante já sinalizado

1. Selecionar o mesmo período.
2. Abrir a turma e disciplina.
3. Filtrar por `Sinalizados`.
4. Abrir o estudante.
5. Ajustar os motivos ou a observação.
6. Salvar novamente.

### Exemplo 3. Corrigir um registro indevido

1. Selecionar o período.
2. Abrir a turma e disciplina.
3. Localizar o estudante.
4. Abrir o registro.
5. Clicar em `Excluir registro`.
6. Confirmar a exclusão.

## 14. Dúvidas e situações comuns

### Não encontro minha turma ou disciplina

Verifique se:

- você escolheu o período correto;
- sua atribuição docente está cadastrada corretamente;
- o período está ativo para uso.

Se a combinação não aparecer, procure a gestão.

### O estudante não aparece na lista

Confira:

- se a turma correta foi selecionada;
- se o filtro está em `Todos`;
- se a busca por nome não está restringindo a lista.

### O sistema não deixa salvar

As causas mais comuns são:

- nenhum estudante selecionado;
- nenhum motivo marcado;
- período fechado para edição.

### O texto do parecer não apareceu

Isso normalmente acontece quando:

- ainda não foi selecionado nenhum motivo;
- o estudante ainda não foi aberto no formulário;
- o período ou a tela ainda não foi carregado corretamente.

## 15. Resumo rápido

Para usar o módulo, o professor deve seguir esta sequência:

1. Entrar em `Pré-Conselho`.
2. Escolher o `Período`.
3. Selecionar a `Turma/Disciplina`.
4. Localizar o estudante.
5. Abrir o `Registro individual`.
6. Marcar os motivos.
7. Informar o nível de atenção.
8. Registrar observação complementar, se necessário.
9. Revisar a pré-visualização.
10. Salvar.

## 16. Encerramento

O módulo de pré-conselho foi pensado para tornar o registro pedagógico mais organizado, objetivo e padronizado. Quanto mais consistentes forem os lançamentos feitos pelos professores, mais útil será a consolidação para a equipe pedagógica e para as reuniões de conselho.
